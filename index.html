Organizador de Tarefas - To-Do List

Sobre o Projeto

Este site apresenta um projeto simples de lista de tarefas (To-Do List) criado utilizando HTML, CSS e JavaScript. O objetivo é ajudar usuários a organizar suas atividades do dia a dia de forma prática e eficiente.

A aplicação permite adicionar tarefas, visualizar o que precisa ser feito e manter um controle básico das atividades. Esse tipo de sistema é muito utilizado para melhorar a produtividade pessoal e profissional.

Características da To-Do List

  • Adicionar novas tarefas facilmente
  • Visualizar tarefas pendentes
  • Organizar atividades do dia

Como utilizar uma lista de tarefas

  1. Escrever todas as tarefas que precisam ser feitas
  2. Organizar as tarefas por prioridade
  3. Marcar as tarefas como concluídas após finalizar
